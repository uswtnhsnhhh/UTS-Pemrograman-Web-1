# Website Toko Buku

Proyek ini merupakan website sederhana untuk menampilkan katalog buku secara online. Pengguna dapat melihat daftar buku lengkap dengan judul, penulis, harga, dan deskripsi singkat. Website ini dirancang dengan tampilan yang responsif dan mudah digunakan.

## Teknologi yang Digunakan
- **HTML5** – untuk struktur halaman
- **CSS3** – untuk desain dan tata letak
- **JavaScript** – untuk interaktivitas sederhana

## Fitur Utama
- Halaman utama berisi daftar buku.
- Tampilan detail setiap buku.
- Navigasi sederhana antar halaman.
- Desain responsif untuk tampilan desktop dan mobile.

## Cara Menjalankan
1. Ekstrak file proyek ke dalam folder lokal.
2. Buka file `index.html` menggunakan browser (misalnya Google Chrome).
3. Website siap digunakan.
